Question dans la vidéo: pour faire un delete de plusieurs authors
/authors:
  delete:
    summary: Delete multiple authors
    operationId: deleteAuthors
    tags:
      - Authors
    requestBody:
      content:
        application/json:
          schema:
            type: array
            items:
              type: string
    responses:
      200:
        description: The list of deleted authors
        content:
          application/json:
            schema:
              type: array
              items:
                $ref: '#/components/schemas/Author'
